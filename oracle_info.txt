OracleToGo 
First you need to have sql developer installed and connect to database ,up and running.
Write your go code
you need to oracle client to be installed in your system
Steps for installing oracle client:
Download the files 
instantclient-basic-macos.x64-12.1.0.2.0.zip
instantclient-sdk-macos.x64-12.1.0.2.0.zip
instantclient-sqlplus-macos.x64-12.1.0.2.0.zip

from oracle website  https://oracle.github.io/odpi/doc/installation.html#macos for help

Steps followed in my system:

i created a folder called Oracle in path :    /Users/divya.p/Documents/  final after creating 
/Users/divya.p/Documents/Oracle
Downloaded the above  three zip files under path /Users/divya.p/Documents/Oracle
instantclient-basic-macos.x64-12.1.0.2.0.zip
instantclient-sdk-macos.x64-12.1.0.2.0.zip
instantclient-sqlplus-macos.x64-12.1.0.2.0.zip
And in the same path : /Users/divya.p/Documents/Oracle created a folder called as
instantclient_12_1  ,/Users/divya.p/Documents/Oracle/instantclient_12_1
Inside this folder /Users/divya.p/Documents/Oracle/instantclient_12_1 , i pasted all other three zip files contents all.
I think this is optional of creating oci8.pc  under this same same path: /Users/divya.p/Documents/Oracle/instantclient_12_1
contents of oci8.pc    ===

prefixdir=/Oracle/instantclient_12_1/sdk
libdir=${prefixdir}
includedir=${prefixdir}/include
   
Name: OCI
Description: Oracle database driver
Version: 11.2
Libs: -L${libdir} -lclntsh
Cflags: -I${includedir}

After this you have to set path :LD_LIBRARY_PATH=/Users/divya.p/Documents/Oracle/instantclient_12_1 in .bash_profile
